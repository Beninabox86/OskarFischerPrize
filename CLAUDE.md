# Oskar Fischer Prize Website

## Project Overview
A React 19 + TypeScript website featuring a living literature review of the 2022 Oskar Fischer Prize winners.

## Tech Stack
- React 19 + TypeScript
- Vite build system
- Tailwind CSS
- GA4 + Microsoft Clarity analytics
- Formbricks for email signups
- Deployed on Vercel (oskarfischerprize.org)
